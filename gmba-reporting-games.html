GMBA - Reporting Game Scores

To enter scores:

  1. Go to http://gmba.ca
  2. Click Admin Login in the top right corner
  3. Log in with your username and password (sent via separate email)
  4. Scroll down to the Statistics Tools section
  5. Select View ALL unreported games or View today's unreported games
  6. For each unreported game, click Report and enter the results
  7. Click Submit Game Report
